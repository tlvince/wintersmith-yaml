# wintersmith-yaml

Reads YAML files and adds them to the content tree.

## Installing

Install globally or locally using `npm`:

```
npm install [-g] wintersmith-yaml
```

and add `wintersmith-yaml` to your config.json

```json
{
  "plugins": [
    "wintersmith-yaml"
  ]
}
```
